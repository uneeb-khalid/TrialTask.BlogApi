# TrialTask.BlogApi


Goal
Define RESTful APIs and business logic for the blogging application.


User Story
› As a user running application can create new Blogs\Posts and add Comments to existed Posts.
› As a user running application can edit own Blogs\Posts.
› As a user running application can view all Blogs\Posts and Comments created.
› As a user running application can view Blogs created during last 10 days.
› As a user running application can view Blogs contain post created during last 30 days.
